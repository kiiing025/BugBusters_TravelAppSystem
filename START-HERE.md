# START HERE

## What this folder is
This folder is your project scaffold.
You will use it as the base folder for your whole capstone/project.
It already matches the structure in your master plan.

## Where to put this folder
1. Download the zip file.
2. Extract it anywhere easy to find.
3. Best place: Desktop or Documents.
4. Rename it only if your teacher wants a different name.

Example:
- Desktop/travel-app-project-template

## What to do after extracting
1. Open the folder in VS Code.
2. Read `SETUP-STEPS.md`.
3. Create a GitHub repository.
4. Push this folder to GitHub.
5. One by one, turn each service folder into a real Lumen app.

## Suggested tools to install first
- VS Code
- PHP
- Composer
- MySQL
- Postman
- Git
- XAMPP or Laragon (optional, but helpful)

## Best working order
1. Finish Jira subtasks and sprint assignments.
2. Finish Figma content.
3. Set up Auth Service.
4. Set up Weather Service.
5. Set up Maps Service.
6. Set up Hotels Service.
7. Set up Payment Service.
8. Set up API Gateway.
9. Test in Postman.
10. Write docs.
11. Deploy.
