# Website Prototype

Open `index.html` in a browser to view the modern website version of the travel app mockup.

This is a static interactive prototype. It includes:

- Dashboard
- Destination search
- Weather and map result
- Hotel listing
- Booking and payment

The prototype follows the same backend story as the API Gateway project:

1. The user searches through the website.
2. The website sends requests to the API Gateway.
3. The gateway calls Auth, Maps, Weather, Hotels, and Payment services.
4. The user completes booking and payment through one flow.
